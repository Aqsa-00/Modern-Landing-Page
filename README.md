# 🚀 AstroFlow – Creative Workflow Automation

AstroFlow is a modern, responsive landing page designed for a creative workflow automation platform. It showcases product features, testimonials, pricing plans, and a call-to-action with smooth animations and interactive UI.

## 🌐 Live Features

- Responsive design (mobile-first)
- Sticky navigation bar with blur (glass effect)
- Animated hero section
- Feature cards with hover effects
- Testimonial slider (auto + manual)
- Pricing section with highlighted plan
- CTA signup form
- ScrollReveal animations
- Hamburger menu for mobile

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Vanilla)
- ScrollReveal.js

## 📁 Project Structure

AstroFlow/

🎯 Key Concepts Implemented
CSS variables for scalable design system
Responsive layouts using Grid & Flexbox
DOM manipulation and event handling
Component-based section structuring
Animation using ScrollReveal
UI/UX best practices (spacing, typography, contrast)
